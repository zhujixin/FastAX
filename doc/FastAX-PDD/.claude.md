# FastAX PDD — 按 domain 拆分版

本目录是 PDD（产品设计文档）的多文件拆分版，按 Go package 组织。

## 使用方式

- **首次阅读**：从 [00-index.md](00-index.md) 开始，获取架构总览和设计原则
- **实现功能**：读取对应 domain 的文件（如实现路由转发 → `06-token-proxy-vendor.md`）
- **查询数据库设计**：`02-database.md` 包含所有表定义
- **查询 API 接口**：`03-api.md` 包含所有端点设计
