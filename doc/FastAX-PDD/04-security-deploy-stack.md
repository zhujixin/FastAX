### 9.2 加密方案

| 场景 | 算法 | 密钥管理 |
|------|------|----------|
| 密码存储 | BCrypt (cost=12) | 盐值内建 |
| 敏感数据存储 | AES-256-GCM | 密钥存于 KMS / 硬件加密机 |
| 传输加密 | TLS 1.3 | Let's Encrypt / 云厂商证书 |
| JWT 签名 | RS256 (RSA 2048) | 私钥存于 KMS |
| API 签名 | HMAC-SHA256 | 用户级 Secret Key |
| 支付回调签名 | 平台自有算法 (微信/支付宝) | 平台密钥 |

```
                             ┌──────────────────┐
                             │   DNS / CDN       │

                             │  · 静态资源       │

                             └────────┬─────────┘
                                      │
                             ┌────────▼─────────┐
                             │  Nginx            │
                             │  · HTTPS 终止      │
                             │  · 静态文件服务     │
                             │  · API 反向代理     │
                             └────────┬─────────┘
                                      │
                             ┌────────▼─────────┐
                             │  fastax-server    │
                             │  Go 单二进制, 2 实例 │
                             │  ┌─────────────┐  │
                             │  │ Gin HTTP    │  │
                             │  │ · 路由+中间件 │  │
                             │  │ · 11 domain  │  │
                             │  │   package    │  │
                             │  │ · Go interface│  │
                             │  │   松耦合调用  │  │
                             │  └─────────────┘  │

                             │  │ · MQ 消费   │  │

                             │  │ · 日志写入  │  │
                             │  └─────────────┘  │
                             └────────┬─────────┘
                                      │
             ┌────────────────────────┼──────────────────────┐
             │                        │                      │
     ┌───────▼───────┐      ┌─────────▼────────┐   ┌───────▼───────┐
     │  SQLite (WAL) │      │  Redis           │   │  RabbitMQ     │
     │  · 单 DB 文件  │      │  · 缓存/会话     │   │  · 异步消息    │
     │  · 所有 domain │      │  · 配额/限流     │   └───────────────┘
```

> **微服务演进**：当注册用户 >5000 或模块 TPS >500 时，将对应 domain 抽取为独立 gRPC 服务。
> domain 的 Service interface 即 proto 定义，抽取时无需修改业务代码。

### 10.2 资源估算 (MVP 阶段)

| 组件 | 规格 | 实例数 | 部署方式 | 备注 |
|------|------|--------|----------|------|
| fastax-server (Go) | 4C8G | 2 | 单二进制 | 11 domain 合并部署 |
| SQLite | SSD | 1 DB 文件 | 嵌入 | WAL 模式 |
| Redis | 2C4G | 1 | 独立 | 缓存+会话+限流 |
| RabbitMQ | 2C4G | 1 | 独立 | 异步解耦 |

## 12. 附录：术语对照

| PRD 术语 | PDD 对应设计 |
|----------|-------------|
| 用户 (User) | user 模块 + GORM 模型 + user_profile 表 (含 preferred_language) |
| 子账号 (SubAccount) | sub_account 表 + RBAC 权限控制 |
| Token 商品 (TokenProduct) | token 模块 + GORM 模型 + Redis 缓存 (含 name_i18n) |
| Token 库存 (TokenInventory) | stock 表 + Redis + 预警调度 |
| 供应商 (Supplier) | supplier 表 + Adaptor 适配器模式 + region 字段 |
| Token 中转调用 | proxy 模块 + SSE 流式代理 + 智能路由 (优先级分组+随机) |
| 渠道健康检测 | 实时检测 (10s) + 定时检测 (5min) + 自动禁用/恢复 |
| **转发逻辑 (ROUTE)** | **Ability 索引 + 优先级分组 + 权重随机 + 轻量熔断 + 跨渠道重试** |
| **智能路由引擎** | **CacheGetRandomSatisfiedChannel + InitChannelCache 内存缓存** |
| **熔断 (Circuit Breaker)** | **轻量自动禁用 (5xx/超时触发), 排除 401/403/429, 健康检测恢复** |
| 风控引擎 (Risk Engine) | risk 模块 + 规则引擎 + 四级预警 (绿/黄/橙/红) |
| 安全护栏 (Guardrail) | guardrail 模块 + PII/注入/密钥检测 + Before/After 两阶段 |
| **多语言通知** | **notification_template 表含 language 字段 + 按用户语言选择模板** |
| 多货币支持 | currency 字段 + 汇率服务 |
| **海外用户邮箱注册** | **user 表 email_verified 字段 + 邮件验证码 + 无手机号注册流程** |
| **国内模型 API 适配** | **VendorAdapter + 请求/响应格式转换层 + OpenAI 兼容输出** |
| **海外→国内专线** | **双轨部署 (海外节点 + 国内节点) + 延迟敏感路由** |
| **跨境合规** | **购买前合规提示弹窗 + 英文用户协议 + 数据出境说明** |
| **供应商入驻 (SUP)** | **vendor-service + supplier_vendor 表 + 审核流程 + 结算体系** |
| **供应商店铺管理** | **供应商门户 (Vendor Portal) + supplier_vendor 信息维护** |
| **供应商商品** | **supplier_product 表 + 多语言字段 + 价格上下限校验** |
| **供应商结算** | **settlement 表 + T+7/月结 + 在线确认 + 提现申请** |
| **供应商 API 健康监控** | **定时健康检测 (30s) + 自动下架 + 通知供应商** |
| **平台佣金** | **commission_rate 字段 + 按交易额百分比计算 + 结算时自动扣除** |
