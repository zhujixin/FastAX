### 5.4 风控模块 (Risk Service)

#### 5.4.1 风控规则引擎架构

```
                  规则引擎架构
┌──────────────────────────────────────────────────┐
│                规则引擎 (EasyRules)                │
│                                                  │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐       │
│  │ 规则集 A  │  │ 规则集 B  │  │ 规则集 C  │       │
│  │ 注册风控  │  │ 交易风控  │  │ API 调用  │       │
│  └──────────┘  └──────────┘  └──────────┘       │
│                                                  │
│  规则格式: IF condition THEN action (level)      │
│                                                  │
│  规则示例:                                       │
│  IF daily_purchase > 50000 THEN freeze(red)      │
│  IF login_city != last_login_city THEN alert     │
│  IF api_calls_per_min > 60 THEN rate_limit       │
└──────────────────────────────────────────────────┘
```

#### 5.4.2 风控等级

| 等级 | 颜色 | 处置动作 | 通知方式 |
|------|------|----------|----------|
| L1 | 绿色 | 仅记录 | 无 |
| L2 | 黄色 | 触发预警，管理员审查 | 站内信 + 邮件 |
| L3 | 橙色 | 限频/限权，临时限制 | 短信 + 邮件 |
| L4 | 红色 | 账号冻结，立即阻断 | 短信 + 电话 |
