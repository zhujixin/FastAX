# FastAX PRD — 按 domain 拆分版

本目录是 PRD 的多文件拆分版。每个文件对应一个 Go package / domain。

## 使用方式

- **首次阅读**：从 [00-index.md](00-index.md) 开始，获取文件索引和功能总览
- **实现功能**：直接读取对应 domain 的文件（如实现路由转发 → `02-token-proxy-vendor.md`）
- **追溯需求 ID**：在 `00-index.md` 的 §2.2 查找需求 ID → 文件映射
- **查询完整清单**：`20-requirements-registry.md` 包含所有需求 ID 的完整注册表
