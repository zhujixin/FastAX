## 7. 功能需求（按模块）

### 6.1 用户模块

#### 6.1.1 注册与登录

| 功能 | 需求描述 | 优先级 | 备注 |
|------|----------|--------|------|
| F-REG-01 | 用户可通过手机号/邮箱+密码+验证码注册 | P0 | 验证码有效期 5 分钟 |
| F-REG-02 | 企业用户注册需上传营业执照等资质，管理员审核后开通 | P1 | 审核超时 48h 自动提醒 |
| F-REG-03 | 支持账号密码登录、短信/邮箱验证码登录 | P0 | 连续 5 次失败暂时锁定 15 分钟 |
| F-REG-04 | 支持微信、支付宝、Google 等第三方 OAuth 登录 | P1 | 首次绑定需绑定手机号 |
| F-REG-05 | 登录成功后生成 JWT Token，有效期 24h，支持刷新机制 | P0 | Token 过期需重新登录 |
| F-REG-06 | 异地登录检测与提醒（短信/邮件通知用户） | P1 | — |
| F-REG-07 | **海外用户邮箱注册**：支持仅邮箱+密码注册，无需手机号，注册后邮件验证 | P0 | 解决海外用户无中国手机号问题 |
| F-REG-08 | **国际验证码**：注册/登录验证码支持邮件发送，适配海外邮箱 | P0 | — |

#### 6.1.2 账号管理

| 功能 | 需求描述 | 优先级 | 备注 |
|------|----------|--------|------|
| F-ACC-01 | 用户可修改个人信息（昵称、头像、手机号、邮箱） | P1 | 修改手机号/邮箱需验证 |
| F-ACC-02 | 用户可重置密码，需身份验证 | P0 | — |
| F-ACC-03 | 用户可绑定/解绑第三方账号 | P2 | — |
| F-ACC-04 | 企业用户可添加子账号、设置子账号权限和 Token 使用限额 | P1 | — |
| F-ACC-05 | 用户可查看当前等级、权限、可使用的功能范围 | P2 | — |
| F-ACC-06 | 管理员可对用户进行等级管理（普通/企业/代理） | P0 | — |
| F-ACC-07 | 管理员可对违规用户进行警告、冻结、注销账号 | P0 | 冻结后用户不可登录和交易 |
