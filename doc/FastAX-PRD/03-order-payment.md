> **Domain**: `domain/order`, `domain/payment` — 订单/支付 | **PDD**: §5.3

### 6.3 交易模块

#### 6.3.1 订单管理

| 功能 | 需求描述 | 优先级 | 备注 |
|------|----------|--------|------|
| F-ORD-01 | 购买/充值 Token 时自动生成订单，含订单号、类型、数量、金额、支付方式、状态 | P0 | 订单号唯一 |
| F-ORD-02 | 订单状态流转：待支付 → 已支付 / 已取消 / 已退款 / 已完成 | P0 | — |
| F-ORD-03 | 待支付订单超时自动取消（超时时间 30 分钟） | P0 | — |
| F-ORD-04 | 用户可申请退款（符合退款规则的订单），管理员审核后完成退款 | P1 | 退款原路返回 |
| F-ORD-05 | 用户可查询所有订单（按订单号、时间、状态筛选） | P0 | — |
| F-ORD-06 | 管理员可查看所有用户订单，支持订单审核和退款处理 | P0 | — |

#### 6.3.2 支付与对账

| 功能 | 需求描述 | 优先级 | 备注 |
|------|----------|--------|------|
| F-PAY-01 | 对接微信支付、支付宝、银行卡支付 | P0 | — |
| F-PAY-02 | 对接 Stripe 支持海外支付（美元/外币结算） | P0 | 海外用户购买国内模型的必需支付方式 |
| F-PAY-03 | 支付过程全程加密，支付成功后系统实时同步状态 | P0 | — |
| F-PAY-04 | 自动生成每日/每月对账报表（交易流水、金额、手续费、退款） | P1 | — |
| F-PAY-05 | 支持对账报表导出（Excel/CSV） | P1 | — |
| F-PAY-06 | **分销佣金系统**：代理用户专属推广链接，按消费额计算佣金 | P2 | 支持佣金提现 |
| F-PAY-07 | 手续费规则配置：明确计算方式和结算周期 | P1 | — |
| F-PAY-08 | **供应商结算管理**：平台按结算周期生成供应商结算单，支持供应商在线确认和提现申请 | P0 | 支持银行转账、PayPal 等 |
| F-PAY-09 | **平台佣金计算**：按供应商费率配置自动计算平台佣金，支持按交易额百分比或固定费率 | P0 | — |
