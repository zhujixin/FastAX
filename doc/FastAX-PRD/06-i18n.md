### 6.9 多语言模块（i18n）

#### 6.9.1 语言检测与切换（LANG-01）

| 功能 | 需求描述 | 优先级 |
|------|----------|--------|
| LANG-01-01 | **浏览器语言自动检测**：用户首次访问时，根据 `Accept-Language` 自动匹配最佳语言 | P0 |
| LANG-01-02 | **语言选择器**：页面顶部/底部固定位置下拉或图标选择器，展示所有可用语言（母语称谓显示，如"中文"、"English"、"日本語"） | P0 |
| LANG-01-03 | **语言持久化**：用户选择的语言存入 `localStorage` / cookie，并同步到用户设置 | P0 |
| LANG-01-04 | **登录态同步**：已登录用户的语言偏好保存到服务端，多设备同步 | P1 |
| LANG-01-05 | **URL 路径语言标识**：可选方案，如 `/en/tokens`、`/ja/tokens`，利于 SEO | P2 |
| LANG-01-06 | **语言回退策略**：当某语言翻译缺失时，自动回退到最近的语言（`zh-TW` → `zh-CN` → `en`） | P0 |
| LANG-01-07 | **语言选择器高亮推荐语言**：根据用户 IP 地域或浏览器语言，在语言列表中标记"推荐" | P1 |

#### 6.9.2 前端 UI 与内容本地化（LANG-02）

| 功能 | 需求描述 | 优先级 |
|------|----------|--------|
| LANG-02-01 | **静态文案翻译**：所有硬编码 UI 文字通过 `t()` 函数或等价机制加载 | P0 |
| LANG-02-02 | **动态文案本地化**：商品名称、分类名称、公告标题等 CMS 内容支持多字段录入 | P0 |
| LANG-02-03 | **占位符与插值**：支持变量替换，如 `"还剩 {days} 天到期"` → `"Expires in {days} days"` | P0 |
| LANG-02-04 | **复数形式**：支持英文等语言的复数规则（`1 item` vs `2 items`） | P1 |
| LANG-02-05 | **日期/时间本地化**：格式随语言变化，中文 `2026-05-27`，英文 `May 27, 2026`，日文 `2026年5月27日` | P0 |
| LANG-02-06 | **数字本地化**：千分位分隔符随 locale 变化（`1,234.56` vs `1 234,56`） | P1 |
| LANG-02-07 | **货币格式本地化**：货币符号位置与格式随 locale（`$1,234` vs `1,234¥` vs `¥1,234`） | P1 |
| LANG-02-08 | **RTL 布局预留**：为未来阿拉伯语等 RTL 语言预留 CSS 逻辑属性方案（如 `margin-inline-start`） | P2 |

#### 6.9.3 通知模板多语言（LANG-03）

| 功能 | 需求描述 | 优先级 |
|------|----------|--------|
| LANG-03-01 | **邮件模板多语言**：注册验证、订单确认、Token 到期、异常提醒等邮件按用户语言发送 | P0 |
| LANG-03-02 | **短信模板多语言**：验证码、安全提醒短信使用用户首选语言 | P0 |
| LANG-03-03 | **站内信多语言**：系统通知、订单通知、风控警告使用用户当前语言 | P0 |
| LANG-03-04 | **模板语言选择策略**：默认按用户设置语言，通知类可附带语言参数覆盖 | P1 |

#### 6.9.4 语言配置（LANG-04）

| 功能 | 需求描述 | 优先级 |
|------|----------|--------|
| LANG-04-01 | **语言列表配置**：管理员可在管理后台启用/禁用语种、设置默认语言 | P0 |
| LANG-04-02 | **语言切换展示**：语言选择器仅显示已启用的语种 | P0 |

#### 6.9.5 开发者文档与 API 多语言（LANG-05）

| 功能 | 需求描述 | 优先级 |
|------|----------|--------|
| LANG-05-01 | **开发者文档多语言**：快速开始、API 参考、SDK 文档支持多语言 | P0 |
| LANG-05-02 | **API 错误消息多语言**：接口响应中的错误信息根据 `Accept-Language` 返回对应语言 | P1 |
| LANG-05-03 | **API 响应多语言字段**：商品详情等接口返回多语言内容（供前端按需渲染） | P1 |
| LANG-05-04 | **代码示例多语言**：文档中的代码注释和说明文字本地化，但代码本身保持英文 | P1 |

#### 6.9.6 帮助中心与客服（LANG-06）

| 功能 | 需求描述 | 优先级 |
|------|----------|--------|
| LANG-06-01 | **FAQ 多语言**：常见问题按语言展示，支持分语种管理 | P1 |
| LANG-06-02 | **工单系统语言标识**：用户提交工单时记录其使用语言，便于分配对应语言客服 | P1 |
| LANG-06-03 | **预置回复多语言**：客服预设回复模板支持多语言 | P2 |
