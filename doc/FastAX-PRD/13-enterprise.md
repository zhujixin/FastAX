### 6.16 企业功能（ENT）

参考 Portkey Enterprise、LiteLLM Proxy 的企业级特性。

| 功能 | 需求描述 | 优先级 | 备注 |
|------|----------|--------|------|
| ENT-01 | **SSO/SAML/OIDC 集成**：支持企业通过 SAML、OIDC、LDAP 进行单点登录 | P2 | — |
| ENT-02 | **团队/项目隔离**：多团队共用平台时，各团队数据隔离，独立计费和限额 | P1 | — |
| ENT-03 | **审计导出**：一键导出操作审计日志，带合规头信息（SOC2/GDPR/HIPAA） | P1 | — |
| ENT-04 | **角色级预算控制**：按角色设定调用限额和预算审批流程 | P2 | — |
| ENT-05 | **预付费套餐**：企业可购买预付费额度和用量承诺优惠 | P1 | — |
| ENT-06 | **API 限速定制**：企业用户可自定义 API Rate Limit 参数 | P1 | — |
| ENT-07 | **模型白名单/黑名单**：企业管理员可限制团队可调用的模型列表 | P1 | — |
| ENT-08 | **数据驻留控制**：企业可指定数据处理的区域偏好（仅国内/仅海外/指定区域） | P2 | — |
